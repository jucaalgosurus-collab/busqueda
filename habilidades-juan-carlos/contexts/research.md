# HJC Research Context

Exploration mode. Read widely before concluding. Don't write code until understanding is clear.

## Research Protocol
1. Understand the domain first
2. Identify key questions
3. Read relevant code/documentation
4. Form hypotheses
5. Test hypotheses with targeted searches
6. Conclude only when evidence supports it

## Always
- Preserve findings in memory/state/active-state.md
- Note decisions and their rationale
- Record what was tried and failed
